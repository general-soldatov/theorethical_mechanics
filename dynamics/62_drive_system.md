# Количество движения системы

Количеством движения системы будем называть векторную величину Q, равную геометрической сумме (главному вектору) количеств движения всех точек системы.
```math
\vec{𝑄}=∑𝑚_𝑘 \vec{𝑣_𝑘}
```
Центр масс системы:
```math
\vec{𝑟_с}=\frac{∑𝑚_𝑘 \vec{𝑟_𝑘}}{∑𝑚_𝑘}
```
Умножим на $∑𝑚_𝑘$:
```math
∑𝑚_𝑘 \vec{𝑟_с}=∑𝑚_𝑘 \vec{𝑟_𝑘}
```
Или
```math
𝑀\vec{𝑟}_с=∑𝑚_𝑘 \vec{𝑟_𝑘}
```
Найдём производную:
```math
𝑀 \frac{𝑑\vec{𝑟_с}}{𝑑𝑡}=∑𝑚_𝑘  \frac{𝑑\vec{𝑟_𝑘}}{𝑑𝑡}
```
Или получим:
```math
𝑀\vec{𝑣_с}=∑𝑚_𝑘 \vec{𝑣_𝑘}
```
Отсюда получим:
```math
𝑀\vec{𝑣_с} =\vec{𝑄}
```
т.е. количество движения системы равно произведению массы всей системы на скорость ее центра масс.

если **центр масс не движется**, то **количество движения равно нулю**. Например, количество движения тела, вращающегося вокруг неподвижной оси, проходящей через его центр масс, будет равно нулю.
Если движение тела является сложным, то величина Q не будет зависеть от его вращательного движения вокруг центра масс.

При сложении дифференциальных уравнений системы точек получим:
```math
∑𝑚_𝑘 \vec{𝑎_𝑘}=∑\vec{𝐹_𝑘^𝑒}+∑\vec{𝐹_𝑘^𝑖}
```
По свойству внутренних сил $∑\vec{𝐹_𝑘^𝑖}=0$, представим $\vec{𝑎_𝑘} в виде $\frac{d\vec{𝑣_𝑘}}{𝑑𝑡}, тогда:
```math
∑𝑚_𝑘 \vec{𝑎_𝑘}=\frac{𝑑}{𝑑𝑡}∑𝑚_𝑘\vec{𝑣_𝑘}
```
С учётом $∑𝑚_𝑘 \vec{𝑣_𝑘}=\vec{𝑄} получим:
```math
\frac{d\vec{𝑄}}{𝑑𝑡}=∑\vec{𝐹_𝑘^𝑒}
```
Производная по времени от количества движения системы равна геометрической сумме всех действующих на систему внешних сил. В проекциях на координатные оси будет:
```math
\frac{d𝑄_𝑥}{𝑑𝑡}=∑𝐹_{𝑘𝑥}^𝑒,{\ \ \ \ }\frac{d𝑄_𝑦}{𝑑𝑡}=∑𝐹_{𝑘𝑦}^𝑒,{\ \ \ \ }\frac{d𝑄_𝑧}{𝑑𝑡}=∑𝐹_{𝑘𝑧}^𝑒
```
Найдем другое выражение этой теоремы.

Пусть в момент времени $t=0$ количество движения системы равно $Q_о$, а в момент $t_1$ становится $Q_1$. Тогда получим:
```math
\vec{𝑄}_1−\vec{𝑄}_0=∑∫_0^{𝑡_1}\vec{𝐹}_𝑘^𝑒 𝑑𝑡
```
Так как интегралы, стоящие справа дают импульсы внешних сил:
```math
∑∫_0^{𝑡_1}\vec{𝐹}_𝑘^𝑒 𝑑𝑡=∑𝑆_𝑘^𝑒
```
Получим:
```math
\vec{𝑄_1}−\vec{𝑄_0}=∑\vec{𝑆_𝑘^𝑒}
```
Получаем, что _изменение количества движения системы за некоторый промежуток времени равно сумме импульсов, действующих на систему внешних сил за тот же промежуток времени_.
Что является **теоремой об изменении количества движения системы в интегральной форме**. В проекциях на координатные оси:
```math
𝑄_{1𝑥}−𝑄_{0𝑥}=∑𝑆_{𝑘𝑥}^𝑒
```
```math
𝑄_{1𝑦}−𝑄_{0𝑦}=∑𝑆_{𝑘𝑦}^𝑒
```
```math
𝑄_{1𝑧}−𝑄_{0𝑧}=∑𝑆_{𝑘𝑧}^𝑒
```
## Закон сохранения количества движения системы
Если сумма всех внешних сил, действующих на систему, равна нулю, то вектор количества движения системы будет постоянен по модулю и направлению:
```math
∑\vec{𝐹_𝑘^𝑒}=0,{\ \ \ }то{\ \ \ }\vec{𝑄}=𝑐𝑜𝑛𝑠𝑡
```
Если сумма проекций всех действующих внешних сил на какую-нибудь ось равна нулю, то проекция количества движения системы на эту ось постоянная величина.
```math
∑𝐹_{𝑘𝑥}^𝑒 =0,{\ \ \ }то{\ \ \ \ } 𝑄_𝑥=𝑐𝑜𝑛𝑠𝑡
```
Эти результаты выражают **закон сохранения количества движения системы**.
## Пример
Пусть треугольник массой M движется вдоль горизонтальной оси $х$ со скоростью $v$. В некоторый момент тело массы m начинает двигаться по наклонной плоскости треугольника с относительной скоростью $u$.

![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/task16.png)

Требуется определить новую скорость треугольника $v_1$. Так как в направлении движения внешние силы отсутствуют, то $𝑀_𝑐$  $𝑣_𝑐=𝑐𝑜𝑛𝑠𝑡$. Отсюда:
```math
(𝑀+𝑚)𝑣=𝑀𝑣_1+𝑚(𝑣_1+𝑢 \cdot cos⁡𝛼 )
```
Раскрывая скобки, получаем:
```math
𝑣_1=𝑣−\frac{𝑚𝑢 cos⁡𝛼}{𝑀+𝑚}
```
Если $𝑚→0$ или $𝑢→0$, то $𝑣 = 𝑣_1$, то есть при малой массе тела или его скорости скорость треугольника практически не меняется. Если $𝑀→∞$, то $𝑣 = 𝑣_1$, то есть при очень большой массе треугольника скорость треугольника также практически не меняется.
Если
```math
𝑣=\frac{𝑚𝑢 \cdot cos⁡𝛼}{𝑀+𝑚}
```
то треугольник остановится.
При достаточно больших значениях относительной скорости $u$, направление движения треугольника может измениться на противоположное.
