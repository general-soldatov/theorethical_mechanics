# Момент инерции тела относительно оси. Радиус инерции
При поступательном движении твердого тела мерой его инертности является **масса**. При вращательном – **момент инерции**.
**Моментом инерции тела** или системы относительно данной оси $О𝑧$ называется скалярная величина, равная сумме произведений масс всех точек тела или системы на квадрат расстояний от этой оси:
```math
𝐽_𝑧=∑𝑚_𝑘 ℎ_𝑘^2
```
Другими словами, **осевой момент инерции** является мерой инертности тела при вращательном движении. При этом для одной материальной точки, находящейся на расстоянии h от оси, $𝐽_𝑧=𝑚ℎ^2$.
Для вычисления осевых моментов инерции можно расстояния от осей выражать через координаты $х_k$, $у_k$, $z_k$ этих точек. Тогда моменты инерции относительно осей $Oxyz$ будут определяться формулами:
```math
𝐽_𝑥=∑𝑚_𝑘 (𝑦_𝑘^2+𝑧_𝑘^2 )
```
```math
𝐽_𝑦=∑𝑚_𝑘 (𝑥_𝑘^2+𝑧_𝑘^2 )
```
```math
𝐽_𝑧=∑𝑚_𝑘 (𝑦_𝑘^2+𝑥_𝑘^2 )
```
![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/meme23.png)

В случае однородного твердого тела, когда количество точек стремиться к бесконечности, а масса точки – к нулю, суммы в формулах выше превращаются в интегралы по соответствующему объему:
```math
𝐽_𝑥=∫(𝑦_𝑘^2+𝑧_𝑘^2 )  𝑑𝑚,{\ \ \ }𝐽_𝑦=∫(𝑥_𝑘^2+𝑧_𝑘^2 )𝑑𝑚
```
```math
𝐽_𝑧=∫(𝑦_𝑘^2+𝑥_𝑘^2 )𝑑𝑚
```
Интегралы в формулах следует понимать как интегралы в обобщенном смысле, так как в некоторых ситуациях они могут обратиться в тройные, двойные и даже в обычные определенные интегралы.
В случае однородных тел $𝑚=𝛾𝑉$, где 𝛾 – плотность тела, V – его объём, т.к. $𝑑𝑚=𝛾𝑑𝑉$, $𝛾=\frac{𝑀}{𝑉}$, где 𝑀 – масса тела, то формулы примут вид:
```math
𝐽_𝑥=\frac{𝑀}{𝑉}∫(𝑦_𝑘^2+𝑧_𝑘^2 )  𝑑𝑉
```
```math
𝐽_𝑦=\frac{𝑀}{𝑉}∫(𝑥_𝑘^2+𝑧_𝑘^2 )𝑑𝑉
```
```math
𝐽_𝑧=\frac{𝑀}{𝑉}∫(𝑦_𝑘^2+𝑥_𝑘^2 )𝑑𝑉
```

## Момент инерции стержня
Например, момент инерции однородного стержня длиной L относительно центральной оси.
![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/task10.png)

Равен определенному интегралу в промежутке от $- L/2$ до $L/2$:
```math
𝐽_𝑧=\frac{𝑀}{𝐿} ∫_{−𝐿/2}^{𝐿/2}𝑥^2 𝑑𝑥=\frac{𝑀}{𝐿}\left(\frac{𝑥^3}{3}\right)_{−𝐿/2}^{𝐿/2}=\frac{𝑀}{𝐿} \frac{𝐿^3}{12}
```
```math
𝐽_𝑧=\frac{𝑀𝐿^2}{12}
```

## Момент инерции однородного диска
Найдём момент инерции сплошного однородного диска радиуса R относительно центральной оси.

![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/task11.png)

Примем, что $𝑑𝑆=𝑟𝑑𝑟𝑑𝜑$, $𝑥^2+𝑦^2=𝑟^2$:
```math
𝑆=∬𝑟𝑑𝑟𝑑𝜑=∫_0^𝑅𝑑𝑟∫_0^R 2𝜋𝑟𝑑𝜑=∫_0^𝑅 2𝜋𝑟𝑑𝑟=𝜋𝑅^2
```
```math
𝐽_𝑧=\frac{𝑀}{𝑉} ∫(𝑦^2+𝑥^2 )𝑑𝑉=\frac{𝑀}{𝑆} ∬𝑟^2\cdot 𝑟\cdot 𝑑𝑟𝑑𝜑
```
```math
𝐽_𝑧=\frac{𝑀}{𝜋𝑅^2} ∫_0^2𝜋𝑑𝜑 ∫_0^𝑅 𝑟^3 𝑑𝑟=\frac{𝑀}{𝜋𝑅^2} ∫_0^2𝜋 \frac{𝑅^4}{4} 𝑑𝜑=\frac{𝑀}{𝜋𝑅^2}\cdot \frac{𝑅^4}{4}\cdot 2𝜋=\frac{𝑀𝑅^2}{2}
```
## Момент инерции прямоугольной пластины
Найдём момент инерции сплошной однородной прямоугольной пластины со сторонами 𝑎 и 𝑏 через ось z, проходящую через центр масс пластины.

![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/task12.png)

Примем, что $𝑑𝑆=𝑑𝑥𝑑𝑦$:
```math
𝑆=∬𝑑𝑥𝑑𝑦=∫_{−𝑎/2}^{𝑎/2}𝑑𝑦 ∫_{−𝑏/2}^{𝑏/2}𝑑𝑥=𝑎\cdot 𝑏
```
```math
𝐽_𝑧=\frac{𝑀}{𝑉} ∫(𝑦^2+𝑥^2)𝑑𝑉=\frac{𝑀}{𝑆} ∬(𝑦^2+𝑥^2)𝑑𝑥𝑑𝑦
```
```math
𝐽_𝑧=\frac{𝑀}{𝑆} ∫_{−𝑎/2}^{𝑎/2}𝑑𝑦 ∫_{−𝑏/2}^{𝑏/2}(𝑦^2+𝑥^2)𝑑𝑥=\frac{𝑀}{𝑆} ∫_{−𝑎/2}^{𝑎/2}\left(𝑥^2 𝑏+\frac{𝑏^3}{12}\right)𝑑𝑦
```
```math
𝐽_𝑧=\frac{𝑀}{𝑆} \frac{𝑎^3 𝑏+𝑎𝑏^3}{12}=\frac{𝑀}{𝑎\cdot 𝑏} \frac{𝑎\cdot 𝑏(𝑎^2+𝑏^2)}{12}=\frac{𝑀}{12}\cdot (𝑎^2+𝑏^2 )
```

## Момент инерции кольца
Найдём момент инерции сплошного однородного диска радиуса R относительно центральной оси. Примем, что $𝑑𝑆=𝑟𝑑𝑟𝑑𝜑$, $𝑥^2+𝑦^2=𝑟^2$:
```math
𝑆=∬𝑟𝑑𝑟𝑑𝜑=∫_{𝑅_1}^{𝑅_2}𝑑𝑟 ∫_0^{2𝜋}𝑟𝑑𝜑=∫_{𝑅_1}^{𝑅_2}2𝜋𝑟𝑑𝑟=𝜋(𝑅_1^2−𝑅_2^2 )
```
```math
𝐽_𝑧=\frac{𝑀}{𝑉} ∫(𝑦^2+𝑥^2 )𝑑𝑉=\frac{𝑀}{𝑆} ∬𝑟^2\cdot 𝑟\cdot 𝑑𝑟𝑑𝜑
```
```math
𝐽_𝑧=\frac{𝑀}{𝑆} ∫_0^{2𝜋}𝑑𝜑 ∫_{𝑅_1}^{𝑅_2}𝑟^3 𝑑𝑟=\frac{𝑀}{𝑆} ∫_0^{2𝜋}𝑑𝜑 \left(\frac{𝑟^4}{4}\right)_{𝑅_1}^{𝑅_2}=\frac{𝑀}{𝑆} ∫_0^{2𝜋}\frac{𝑅_1^4−𝑅_2^4}{4} 𝑑𝜑
```
```math
𝐽_𝑧=\frac{𝑀}{𝜋(𝑅_1^2−𝑅_2^2 )}\cdot 2𝜋 \cdot \frac{(𝑅_1^2−𝑅_2^2 )(𝑅_1^2+𝑅_2^2 )}{4}=\frac{𝑀}{2}\cdot(𝑅_1^2+𝑅_2^2 )
```
