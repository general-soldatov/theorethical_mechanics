# Колебательные процессы
Допустим, что точка $M$ движется прямолинейно под действием только восстанавливающей силы:
```math
𝐹_𝑥=−𝑐𝑥
```
Закон движения будет выглядеть следующим образом:
```math
𝑚\ddot{𝑥}=𝐹_𝑥=−𝑐𝑥
```
Зная, что циклическая частота колебаний определяется по формуле:
```math
𝜔=\sqrt{\frac{c}{𝑚}}
```
Получим уравнение следующего вида:
```math
\ddot{𝑥}+𝜔^2 𝑥=0
```
Данное уравнение является **дифференциальным уравнением свободных колебаний** при отсутствии сопротивления.

![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/meme7.png)

Поскольку это уравнение с постоянными коэффициентами, то решение уравнения ищется в виде $𝑥=𝑒^{𝑘𝑡}$, где $𝑘$ – некоторое число.
Подставим эту функцию в дифференциальное уравнение и получим алгебраическое уравнение 2-го порядка:
```math
𝑘^2+𝜔^2=0
```
Это уравнение называется **характеристическим** уравнением для дифференциального уравнения. Корень этого уравнения является комплексным:
```math
𝑘=\sqrt{−𝜔^2}=𝜔\sqrt{−1}=𝜔𝑖
```
Согласно теореме, если корни характеристического уравнения комплексные вида $𝑘_{1,2}=𝛼±𝛽𝑖$, то решение примет вид:
```math
𝑥=𝑒^{𝛼𝑡} (𝐶_1  cos⁡𝛽𝑡+𝐶_2  sin⁡𝛽𝑡)
```
В нашем случае решение примет вид:
```math
𝑥=𝐶_1  cos⁡𝜔𝑡+𝐶_2  sin⁡𝜔𝑡
```
Где $𝐶_1$ и $𝐶_2$ - произвольные постоянные.
Величина $𝜔$ является частотой колебаний.
Период колебаний - $𝑇=\frac{2𝜋}{𝜔}$
Введём новые постоянные $𝐶_1=𝐴 sin⁡𝜑$ и $𝐶_2=𝐴 cos⁡𝜑$. Получим:
```math
𝑥=𝐴 sin⁡𝜑 cos⁡𝜔𝑡+𝐴 cos⁡𝜑 sin⁡𝜔𝑡
```
Зная формулу синуса суммы двух аргументов:
```math
sin⁡(𝛼+𝛽)=sin⁡𝛼  cos⁡𝛽+sin⁡𝛽  cos⁡𝛼
```
Проведём преобразования:
```math
𝑥=𝐴 sin⁡(𝜔𝑡+𝜑)
```
Таким образом мы получили общее решение дифференциального уравнения свободных колебаний.

### Задача Коши

Для определения постоянных $𝐴$ и $𝜑$ зададим начальные условия: $𝑡=0$, $𝑥=𝑥_0$, $\dot{𝑥}=𝑣_0$. Учитывая, что \dot{𝑥}=𝐴𝜔 cos(𝜔𝑡+𝜑), Получаем:
```math
\begin{cases} 𝑥_0=𝐴 sin⁡(0\cdot 𝜔+𝜑)=𝐴 sin⁡𝜑 \\ 𝑣_0=𝐴𝜔 cos⁡𝜑 \end{cases}
```
Разделим 1 уравнение на 2:
```math
\frac{𝑥_0}{𝑣_0} =\frac{𝐴 sin⁡𝜑}{𝐴 𝜔 cos⁡𝜑}  ⇒  tg⁡𝜑=\frac{𝑥_0 𝜔}{𝑣_0}
```
Зная основное тригонометрическое тождество $sin^2⁡𝜑+cos^2⁡𝜑=1$, выразим $cos⁡𝜑$ и $sin⁡𝜑$:
```math
sin⁡𝜑=\frac{𝑥_0}{𝐴}{,\ \ \ }cos⁡𝜑=\frac{𝑣_0}{𝐴𝜔}
```
Проведём преобразования:
```math
\frac{𝑥_0^2}{𝐴^2} +{𝑣_0^2}{𝐴^2 𝜔^2}=1⇒ 𝐴=\sqrt{𝑥_0^2+\frac{𝑣_0^2}{𝜔^2}}
```
Где $𝐴$ – амплитуда и $𝜑$ – начальная фаза колебаний.
