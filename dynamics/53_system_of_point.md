# Радиус инерции
В случае, если $𝑅_1→𝑅_2$, то формула момента инерции для кольца приобретёт следующий вид:
```math
𝐽_𝑧=\frac{𝑀}{2}\cdot\left(𝑅^2+𝑅^2 \right)=𝑀𝑅^2
```
Часто в ходе расчетов пользуются понятием **радиуса инерции**. Это линейная величина, определяемая равенством:
```math
𝜌_𝑧=\sqrt{\frac{𝐽_𝑧}{𝑀}}
```
Радиус инерции геометрически равен расстоянию от оси $Oz$ той точки, в которой надо сосредоточить массу всего тела, чтобы момент инерции одной этой точки был равен моменту инерции всего тела.

Зная радиус инерции и массу соответствующего тела, его момент инерции определяется по формуле:
```math
𝐽_𝑧=𝑀𝜌_𝑧^2
```

## Моменты инерции тела относительно параллельных осей
Моменты инерции данного тела относительно различных осей будут разными.
![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/task14.png)

Покажем, как, зная момент инерции тела относительно одной координатной оси, найти момент инерции относительно другой, ей параллельной. Оси $x$ пересекаются, остальные – параллельны друг другу. Получим:
```math
𝐽_{𝑜𝑧}=∑𝑚_𝑘 (𝑥_𝑘^2+𝑦_𝑘^2 )
```
```math
𝐽_{𝑐𝑧1}=∑𝑚_𝑘 (𝑥_1𝑘^2+𝑦_1𝑘^2 )
```
Из рисунка видно, что $𝑥_𝑘=𝑥_{1𝑘}−𝑑$ и $𝑥_𝑘^2=𝑥_{1𝑘}^2−2𝑥_{1𝑘} 𝑑+𝑑^2$, а $𝑦_𝑘=𝑦_{1𝑘}$. Подставим эти значения в формулу:
```math
𝐽_{𝑜𝑧}=∑𝑚_𝑘 (𝑥_{1𝑘}^2−2𝑥_{1𝑘} 𝑑+𝑑^2+𝑦_{1𝑘}^2 )
```
```math
𝐽_{𝑜𝑧}=∑𝑚_𝑘 (𝑥_{1𝑘}^2+𝑦_{1𝑘}^2 )+∑(𝑚_𝑘 ) 𝑑^2−∑(𝑚_𝑘 𝑥_{1𝑘} )2𝑑
```
Здесь $∑𝑚_𝑘 (𝑥_{1𝑘}^2+𝑦_{1𝑘}^2 )=𝐽_{𝑐𝑧1}$, $∑(𝑚_𝑘 )=𝑀$. Найдём $∑(𝑚_𝑘 𝑥_{1𝑘} )$.
На основании формулы:
```math
𝑥_{1𝑐}=\frac{∑𝑚_𝑘 𝑥_1𝑘}{∑𝑚_𝑘 }
```
Получим:
```math
𝑥_{1𝑐}∑𝑚_𝑘=∑𝑚_𝑘 𝑥_{1𝑘}⇒𝑀𝑥_{1𝑐}=∑𝑚_𝑘 𝑥_{1𝑘}
```
Поскольку $𝑥_𝑐=0$, то $∑𝑚_𝑘 𝑥_{1𝑘}=0$, отсюда следует:
```math
𝐽_{𝑜𝑧}=𝐽_{𝑐𝑧1}+𝑀𝑑^2
```

## Теорема Штейнера-Гюйгенса
Момент инерции тела относительно данной оси равен моменту инерции относительно оси, ей параллельной, проходящей через центр масс тела, сложенному с произведением массы всего тела на квадрат расстояния между осями.

![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/task14.png)

Найдём момент инерции для стержня, относительно оси $z$ по теореме Штейнера-Гюйгенса:
```math
𝐽_𝑧=\frac{𝑚𝐿^2}{12}+𝑚\left(\frac{𝐿}{2}\right)^2=\frac{𝑚𝐿^2}{12}+\frac{𝑚𝐿^2}{4}=\frac{𝑚𝐿^2}{3}
```
