# Кинетическая энергия системы

**Кинетической энергией системы** называется скалярная величина $T$, равная сумме кинетических энергий всех точек системы:
```math
𝑇=∑\frac{𝑚_𝑘 𝑣_𝑘^2}{2}
```
Отличие кинетической энергии от введенных ранее понятий количества движения системы и кинетического момента заключается в том, что
- во-первых, она является характеристикой и поступательного и вращательного движения
- во-вторых, это величина скалярная
- в третьих, на ее изменение влияют как внешние, так и внутренние силы системы.

## Кинетическая энергия поступательного движения

В этом случае все точки тела движутся с одинаковыми скоростями, равными скорости центра масс системы. Следовательно, для любой точки c $𝑣_𝑘=𝑣_𝑐$ и из формулы:
```math
𝑇=∑\frac{𝑚_𝑘 𝑣_𝑐^2}{2}=∑(𝑚_𝑘 )  \frac{𝑣_𝑐^2}{2}=𝑀 \frac{𝑣_𝑐^2}{2}
```
**Кинетическая энергия тела при поступательном движении** равна половине произведения массы тела на квадрат скорости его центра масс.

![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/meme33.png)

## Кинетическая энергия вращательного движения

Если тело вращается вокруг какой-нибудь оси $Oz$, то скорость любой ее точки определяется как $𝑣_𝑘=𝜔ℎ_𝑘$, где $ℎ_𝑘$ – расстояние от точки до оси вращения, $ω$ – угловая скорость тела. Подставляя это значение в формулу, получим
```math
𝑇=∑\frac{𝑚_𝑘 (𝜔ℎ_𝑘 )^2}{2}=∑(𝑚_𝑘 ℎ_𝑘^2 ) \frac{𝜔^2}{2}
```
а учитывая, что $∑(𝑚_𝑘 ℎ_𝑘^2 )=𝐽_𝑧$, окончательно получаем:
```math
𝑇_{𝑟𝑜𝑡}=\frac{𝐽_𝑧 𝜔^2}{2}
```
**Кинетическая энергия тела при вращательном движении** равна половине произведения момента инерции тела относительно оси вращения на квадрат его угловой скорости.

## Кинетическая энергия при плоском движении
При этом движении скорости всех точек тела в каждый момент времени распределены так, как если бы тело вращалось вокруг оси, перпендикулярной плоскости движения и проходящей через мгновенный центр скоростей $Р$.

![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/task20.png)

Следовательно кинетическая энергия будет иметь вид:
```math
𝑇_{плоск}=\frac{𝐽_𝑝 𝜔^2}{2}
```
где $𝐽_𝑝$ - момент инерции относительно оси, проходящей через мгновенный центр скоростей $Р$.

Очевидно, что величина $𝐽_𝑝$ является мгновенной. Введем вместо нее постоянный момент инерции $𝐽_𝑐$ относительно оси, проходящей через центр масс $С$ тела. Тогда по теореме **Штейнера-Гюйгенса**:
```math
𝐽_𝑝=𝐽_𝑐+𝑀𝑑^2
```
Подставим это значение в формулу, с учётом $𝑑=𝑃𝐶$:
```math
𝑇_{плоск}=\frac{(𝐽_𝑐+𝑀𝑑^2)𝜔^2}{2}=\frac{𝐽_𝑐 𝜔^2}{2}+\frac{𝑀𝑑^2 𝜔^2}{2}
```
где $𝐽_𝑝$ - момент инерции относительно оси, проходящей через мгновенный центр скоростей $Р$.

Очевидно, что величина $𝐽_𝑝$ является мгновенной. Введем вместо нее постоянный момент инерции $𝐽_𝑐$ относительно оси, проходящей через центр масс $C$ тела.

Учитывая, что точка $P$ – мгновенный центр скоростей и, следовательно, $𝜔𝑑=𝜔\cdot 𝑃𝐶=𝑣_𝑐$, а, значит:
```math
𝑇_{плоск}=\frac{𝐽_𝑐 𝜔^2}{2}+\frac{𝑀𝑣_𝑐^2}{2}
```
т.е., при **плоском движении кинетическая энергия тела** равна энергии поступательного движения со скоростью центра масс, сложенной с кинетической энергией вращательного движения вокруг центра масс.

### Пример
Определим кинетическую энергию катка массы $M$ и радиуса $R$, движущегося со скоростью $𝑣_с$ по горизонтальной поверхности.

![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/task21.png)

Тогда его кинетическая энергия определится по формуле:
```math
𝑇_{плоск}=\frac{𝐽_𝑐 𝜔^2}{2}+\frac{𝑀𝑣_𝑐^2}{2}
```
Учитывая, что момент инерции сплошного диска:
```math
𝐽_𝑐=\frac{𝑀𝑅^2}{2}
```
Получим:
```math
𝑇_{плоск}=\frac{𝑀𝑅^2 𝜔^2}{4}+\frac{𝑀𝑣_𝑐^2}{2}
```
И с учётом:
```math
𝜔=\frac{𝑣_𝑐}{𝑅}
```
Получим:
```math
𝑇_{плоск}=\frac{𝑀𝑅^2 𝑣_𝑐^2}{4𝑅^2}+\frac{2𝑀𝑣_𝑐^2}{4}=\frac{3}{4} 𝑀𝑣_𝑐^2
```

## Теорема об изменении кинетической энергии системы

Если рассмотреть какую-нибудь точку системы массой $𝑚_𝑘$, имеющую скорость $𝑣_𝑘$, то для этой точки будет:
```math
𝑑 \frac{𝑚_𝑘 𝑣_𝑘^2}{2}=𝑑𝐴_𝑘^𝑒+𝑑𝐴_𝑘^𝑖
```
Где $𝑑𝐴_𝑘^𝑒$ и $𝑑𝐴_𝑘^𝑖$ - элементарные работы действующих на точку внешних и внутренних сил. Для каждой из точек системы найдем, что:
```math
𝑑𝑇=∑𝑑𝐴_𝑘^𝑒+∑𝑑𝐴_𝑘^𝑖
```
Равенство выражает **теорему об изменении кинетической энергии системы в дифференциальной форме**. Проинтегрируем обе части этого равенства:
```math
∫_{𝑇_0}^{𝑇_1}𝑑𝑇=𝑇_1−𝑇_0=∑𝐴_𝑘^𝑒+∑𝐴_𝑘^𝑖
```

### Пример
Определим угловую скорость и угловое ускорение стержня массой $M$ и длиной $L$, движущегося из состояния покоя.

![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/task22.png)

Так как стержень, находящийся в состоянии покоя, совершает вращательное движение относительно оси $z$, проходящей через точку $O$, то теорема об изменении кинетической энергии будет выглядеть следующим образом:
```math
\frac{𝐽_𝑧 𝜔^2}{2}=𝑀𝑔 \frac{𝐿}{2} sin⁡𝜑
```
То есть работа силы тяжести определяется на вертикальном перемещении $\frac{𝐿}{2} sin⁡𝜑$. Учитывая что
```math
𝐽_𝑧=\frac{1}{3} 𝑀𝐿^2
```
Проведём преобразования:
```math
\frac{𝑀𝐿^2 𝜔^2}{3\cdot 2}=𝑀𝑔 \frac{𝐿}{2}sin⁡𝜑{\ \ \ }⇒{\ \ \ \ }\frac{𝐿𝜔^2}{3}=𝑔 sin⁡𝜑
```
Отсюда выразим скорость:
```math
𝜔^2=\frac{3𝑔}{𝐿} sin⁡𝜑{\ }⇒{\ }𝜔=\sqrt{\frac{3𝑔}{𝐿} sin⁡𝜑}
```
Дифференцируя выражение по времени с учетом того, что и $ω$ и $φ$ являются функциями, зависящими от времени, получаем:
```math
2𝜔\cdot \dot{𝜔}=\frac{3𝑔}{𝐿}  cos⁡𝜑 \dot{𝜑}
```
С учётом $\dot{𝜔}=𝜀$, $\dot{𝜑}=𝜔$ получим:
```math
𝜀=\frac{3𝑔}{𝐿} cos⁡𝜑
```
Отметим, что при $φ=π/2$ угловая скорость стержня максимальна, а угловое ускорение равно нулю.

### Пример

Рассмотрим физический маятник.

![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/task23.png)

Уравнение вращения твёрдого тела имеет вид:
```math
𝐽_𝑧 𝜀=∑𝑚_𝑧 (\vec{𝐹_𝑘})=−𝑚𝑔\cdot 𝑂𝐶\cdot sin⁡𝜑
```
С учётом теоремы Штейнера-Гюйгенса получим:
```math
(𝐽_𝑐+𝑚\cdot 𝑂𝐶^2 )\cdot \ddot{𝜑}=−𝑚𝑔\cdot 𝑂𝐶\cdot sin⁡𝜑
```
Где $𝐽_𝑐$ - момент инерции тела относительно оси, проходящей через центр масс $C$. Выразим его через радиус инерции $𝐽_𝑐=𝑚𝜌_𝑐^2$, тогда:
```math
(𝑚𝜌_𝑐^2+𝑚\cdot 𝑂𝐶^2 )\cdot \ddot{𝜑}=−𝑚𝑔\cdot 𝑂𝐶\cdot sin⁡𝜑
```
При малых колебаниях $sin⁡𝜑≈𝜑$, отсюда
```math
\ddot{𝜑}+\frac{𝑔}{(𝜌_𝑐^2+𝑂𝐶^2)/𝑂𝐶} 𝜑=0
```
Приведённая длина физического маятника находится из соотношения:
```math
𝐿_{пр}=\frac{𝜌_𝑐^2+𝑂𝐶^2}{𝑂𝐶}
```
Отсюда получим:
```math
\ddot{𝜑}+\frac{𝑔}{𝐿_{пр}} 𝜑=0
```
Это уравнение абсолютно аналогично уравнению колебаний математического маятника. Отсюда следует:
```math
𝑇=2𝜋\sqrt{\frac{𝐿_{пр}}{𝑔}}
```
Таким образом, колебания данного физического маятника будут происходить по такому же закону, что и колебания математического маятника с длиной нити, равной $𝐿_{пр}$
