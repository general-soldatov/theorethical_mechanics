# Главный момент количеств движения системы (кинетический момент)

**Кинетическим моментом системы** относительно данного центра О называется величина $K_o$, равная геометрической сумме моментов количеств движения всех точек системы относительно этого центра:
```math
\vec{𝐾}_0=∑\vec{𝑚_𝑜}(𝑚_𝑘\vec{𝑣_𝑘})
```
Кинетический момент системы можно рассматривать как характеристику ее вращательного движения.
Найдем кинетический момент вращающегося тела вокруг неподвижной оси z. У любой точки тела, отстоящей от оси вращения на расстояние $ℎ_к$, скорость $𝑣_𝑘=𝜔ℎ_𝑘$. Тогда:
```math
\vec{𝑚_𝑧}(𝑚_𝑘 \vec{𝑣_𝑘})=𝑚_𝑘 𝑣_𝑘 ℎ_𝑘=𝑚_𝑘 𝜔 ℎ_𝑘^2
```

![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/task17.png)

Кинетический момент всего тела будет равен сумме моментов количеств движения всех точек:
```math
\vec{𝐾_𝑧}=∑\vec{𝑚_𝑧}(𝑚_𝑘 \vec{𝑣_𝑘})=\left(∑𝑚_𝑘 ℎ_𝑘^2\right)𝜔
```
Принимая во внимание, что величина $∑𝑚_𝑘 ℎ_𝑘^2=𝐽_𝑧$ окончательно находим:
```math
𝐾_𝑧=𝐽_𝑧 𝜔
```
Кинетический момент вращающегося тела относительно оси вращения равен произведению момента инерции тела относительно этой оси на угловую скорость тела.

## Теорема об изменении кинетического момента системы

Производная по времени от главного момента количеств движения системы относительно некоторого неподвижного центра равна сумме моментов всех внешних сил системы относительно того же центра.

Для одной точки системы верно следующее равенство
```math
\frac{𝑑}{𝑑𝑡} \vec{𝑚_𝑜}(𝑚_𝑘 \vec{𝑣_𝑘})=\vec{𝑚_𝑜}\left(\vec{𝐹_𝑘^𝑒}\right)+\vec{𝑚_𝑜}\left(\vec{𝐹_𝑘^𝑖}\right)
```
тогда для всей системы получим
```math
\frac{𝑑}{𝑑𝑡} \left[∑\vec{𝑚_𝑜}(𝑚_𝑘 \vec{𝑣_𝑘}) \right]=∑\vec{𝑚_𝑜}(\vec{𝐹_𝑘^𝑒})+∑\vec{𝑚_𝑜}(\vec{𝐹_𝑘^𝑖})
```
Поскольку $∑\vec{𝑚_𝑜}(\vec{𝐹_𝑘^𝑖})=0$,  а $\vec{𝐾_0}=∑\vec{𝑚_𝑜}(𝑚_𝑘 \vec{𝑣_𝑘})$, получим:
```math
\frac{𝑑\vec{𝐾_0}}{𝑑𝑡}=∑\vec{𝑚_𝑜}(\vec{𝐹_𝑘^𝑒})
```
В случае вращения твердого тела относительно оси z, когда все силы, действующие на него, являются внешними, то
```math
\frac{𝑑(𝐽_𝑧 𝜔)}{𝑑𝑡}=∑𝑚_𝑧 (\vec{𝐹_𝑘^𝑒})
```
или
```math
𝐽_𝑧 𝜀=∑𝑚_𝑧 (\vec{𝐹_𝑘^𝑒})
```

## Закон сохранения кинетического момента

Если $∑\vec{𝑚_𝑜}(\vec{𝐹_𝑘^𝑒})=0$, $\vec{𝐾_0}=𝑐𝑜𝑛𝑠𝑡$, т.е. если сумма моментов относительно данного центра всех приложенных к системе внешних сил равна нулю, то кинетический момент системы относительно этого центра будет постоянен и по модулю и по направлению.

Если $∑𝑚_𝑧 (\vec{𝐹_𝑘^𝑒})=0$, то $\vec{𝐾_𝑧}=𝑐𝑜𝑛𝑠𝑡$ т.е. если сумма моментов всех действующих на систему внешних сил относительно какой-либо оси равна нулю, то кинетический момент относительно этой оси будет величиной постоянной.

если система абсолютно твердая, то ее $𝐽_𝑧=𝑐𝑜𝑛𝑠𝑡$, а следовательно постоянна и ее угловая скорость.

если система изменяема, то под действием внутренних сил отдельные ее точки могут удаляться от оси, что вызывает увеличение $𝐽_𝑧$, или приближаться к оси, что приводит к уменьшению $𝐽_𝑧$.

Но, так как $𝐽_𝑧 𝜔=𝑐𝑜𝑛𝑠𝑡$, то в первом случае 𝜔 будет уменьшаться, а во втором увеличиваться.

### Пример

Рассмотрим следующий пример.

![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/task17.png)

Пусть однородный диск радиуса $R$ и массой $M=2m$ вращается вокруг вертикальной оси $z$ с угловой скоростью ω. В некоторый момент тело массы m начинает двигаться вдоль обода диска с относительной скоростью.

Требуется определить новую угловую скорость диска $ω1$.

Так как моменты внешних сил относительно оси $z$ равны нулю, то
```math
\vec{𝐾_𝑧}=𝑐𝑜𝑛𝑠𝑡
```
Это означает, что кинетический момент системы до начала движения тела равен кинетическому моменту системы при движении тела:
```math
𝐽_𝑧 𝜔+𝑚𝜔𝑅^2=𝐽_𝑧 𝜔_1+𝑚(𝑣+𝜔_1 𝑅)𝑅
```
В последнем слагаемом последнего уравнения присутствует переносная скорость тела $𝜔_1 𝑅$.

Учитывая, что
```math
𝐽_𝑧=\frac{𝑀𝑅^2}{2}{\ \ \ \ }и{\ \ \ \ \ } 𝑀=2𝑚
```
Получим:
```math
𝑚𝑅^2 𝜔+𝑚𝜔𝑅^2=𝑚𝑅^2 𝜔_1+𝑚𝑣𝑅+𝑚𝜔_1 𝑅^2
```
Или после преобразований:
```math
2𝑅𝜔=2𝑅𝜔_1+𝑣   →   𝜔_1=𝜔−\frac{𝑣}{2𝑅}
```
Таким образом, угловая скорость диска уменьшается, а при $𝑣=2𝜔𝑅$ диск остановится. Более того, при $𝑣>2𝜔𝑅$ диск начнётся вращаться в обратном направлении.
