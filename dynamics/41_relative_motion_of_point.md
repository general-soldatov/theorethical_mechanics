# Относительное движение точки

Рассмотрим основной закон динамики
```math
𝑚\vec{𝑎}=\sum \vec{𝐹_𝑖}
```
Это равенство справедливо в неподвижной системе координат. Однако часто бывает необходимо определять характер относительного движения точки, поэтому, используя теорему Кориолиса, представим основной закон динамики в виде:
```math
𝑚(\vec{𝑎}_{отн}+\vec{𝑎}_{пер}+\vec{𝑎}_{кор})=\sum \vec{𝐹}_𝑖
```
Иначе говоря:
```math
𝑚\vec{𝑎}_{отн} = \sum \vec{𝐹}_𝑖 - m\vec{𝑎}_{пер} - m\vec{𝑎}_{кор}
```
Векторы $𝑚\vec{𝑎}_{пер}$ и $𝑚\vec{𝑎}_{кор} называют соответственно **переносной** и **кориолисовой** силами инерции.
Отметим, что силы инерции появляются только в уравнениях относительного движения точки.

## Пример
Частица движется по лопатке центробежного разбрасывателя (вид сверху), представляющего собой диск, вращающийся с угловой скоростью ω. Указанную лопатку будем принимать за ось $x$ подвижной системы координат, жестко связанной с диском.
![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/task5.png)

На точку действуют следующие силы:
* сила трения о поверхность диска $𝐹_{тр2}=𝑓𝑁_2$,
* нормальная реакция поверхности диска (направлена к нам) $𝑁_2=𝑚𝑔$,
* сила трения о поверхность лопатки $F_{тр1}=𝑓𝑁_1$,
* нормальная реакция $𝑁_1=2𝑚𝜔𝑣_{отн}=2𝑚𝜔\dot{𝑥}$.

Здесь $2𝑚𝜔\dot{𝑥}$ – сила Кориолиса.

Также на точку действует центробежная сила $𝐹_ц=𝑚𝜔^2 𝑥$
Таким образом уравнение относительного движения примет вид:
```math
𝑚\ddot{𝑥}=𝑚𝜔^2 𝑥−2𝑚𝑓𝜔\dot{𝑥}−𝑚𝑔𝑓
```
Или преобразованное уравнение:
```math
\ddot{𝑥}+2𝑓𝜔\dot{𝑥}−𝜔^2 𝑥=−𝑔𝑓
```
Это линейное, неоднородное дифференциальное уравнение второго порядка с постоянными коэффициентами. Его решение, складывается из общего решения однородного и частного решения неоднородного уравнения.
Получим общее решение, задав характеристическое уравнение:
```math
𝑘^2+2𝑓𝜔𝑘−𝜔^2=0
```
Найдём корни уравнения:
```math
𝑘=−𝑓𝜔±𝜔\sqrt{𝑓^2+1}
```
Помним, что корни уравнения ищутся в виде $𝑥=𝐶𝑒^{𝑘𝑡}$.
Соответственно, решение однородного уравнения принимает вид:
```math
𝑥_{одн}=𝐶_1 𝑒^{𝑡𝜔(−𝑓+\sqrt{𝑓^2+1})} +𝐶_2 𝑒^{𝑡𝜔(−𝑓−\sqrt{𝑓^2+1})}
```
Частное решение уравнения имеет вид:
```math
𝑥_ч=\frac{𝑓𝑔}{𝜔^2}
```
Общее решение неоднородного уравнения:
```math
𝑥(𝑡)=𝐶_1 𝑒^{𝑡𝜔(−𝑓+\sqrt{𝑓^2+1})} +𝐶_2 𝑒^{𝑡𝜔(−𝑓−\sqrt{𝑓^2+1})} +\frac{𝑓𝑔}{𝜔^2}
```
Принимаем, что
```math
𝑘_1=𝜔\left(−𝑓+\sqrt{𝑓^2+1}\right)
```
```math
𝑘_2=𝜔\left(−𝑓−\sqrt{𝑓^2+1}\right)
```
тогда уравнение примет вид.
```math
𝑥(𝑡)=𝐶_1 𝑒^{𝑡𝑘_1}+𝐶_2 𝑒^{𝑡𝑘_2}+\frac{𝑓𝑔}{𝜔^2}
```
Дифференцируя это равенство, получаем относительную скорость движения точки:
```math
𝑣_{отн}(𝑡)=𝐶_1 𝑘_1 𝑒^{𝑡𝑘_1}+𝐶_2 𝑘_2 𝑒^{𝑡𝑘_2}
```
Для определения постоянных коэффициентов $C_1$ и $C_2$ введём начальные условия:
```math
𝑡=0,{\ \ \ \ }𝑥=𝑥_0,{\ \ \ } 𝑣_{отн}=0
```
Здесь $х_0$ – начальное положение точки на лопатке, а равенство нулю начальной относительной скорости справедливо в случае вертикального попадания точки на диск.

Подставляя эти начальные условия в уравнения по определению координаты и скорости, получаем:
```math
𝑥_0=𝐶_1+𝐶_2+\frac{𝑓𝑔}{𝜔^2},{\ \ \ }0=𝐶_1 𝑘_1+𝐶_2 𝑘_2
```
Откуда выразим
```math
𝐶_1=\frac{−𝑘_2 \left(𝑥_0−\frac{𝑓𝑔}{𝜔^2}\right)}{𝑘_1−𝑘_2}
```
```math
𝐶_2=\frac{−𝑘_1 \left(𝑥_0−\frac{𝑓𝑔}{𝜔^2} \right)}{𝑘_1−𝑘_2}
```
Подставляя в уравнение координаты вместо х длину лопатки R, можно определить время движения по ней до момента схода с диска, а из формулы зависимости скорости от времени – скорость схода частицы с диска.
