# Работа силы

Для характеристики действия, оказываемого силой на тело при некотором его перемещении, используется понятие **работы силы**.
Элементарная работа
```math
𝑑𝐴=𝐹𝑑𝑠 cos⁡𝛼
```
Если 𝛼 – острый, то работа положительна, если тупой - отрицательна, если α = 90° – работа равна нулю.

![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/task3.png)

Работа силы на любом перемещении $М_оМ_1$
```math
𝐴_{𝑀_0 𝑀_1}=\int_{𝑀_0}^{𝑀_1}\left(\vec{𝐹}∙𝑑\vec{𝑠}\right) =\int_{𝑀_0}^{𝑀_1} 𝐹 cos⁡𝛼 𝑑𝑠 =\int_{𝑀_0}^{𝑀_1} 𝐹_𝑥 𝑑𝑥+𝐹_𝑦 𝑑𝑦+𝐹_𝑧 𝑑𝑧
```
Работа силы на любом перемещении МоМ1 равна взятому вдоль этого перемещения интегралу от элементарной работы. Если $𝐹 cos⁡𝛼=𝑐𝑜𝑛𝑠𝑡$, то
```math
𝐴_{𝑀_0 𝑀_1}=𝐹 𝑠_{𝑀_0 𝑀_1} cos⁡𝛼
```
где s – перемещение $М_оМ_1$. Такой случай возможен, если сила F постоянна по модулю и направлению, а перемещение точки прямолинейно.
Единица измерения работы – 1 джоуль=1Н·м.

## Мощность
Мощностью называется величина, определяющая работу, совершаемую силой в единицу времени.
Если работа совершается равномерно, то мощность:
```math
𝑁=\frac{𝐴}{𝑡}=𝐹 𝑣 cos⁡𝛼
```
где t – время, в течение которого совершалась работа. В общем случае:
```math
𝑁=\frac{𝑑𝐴}{𝑑𝑡}=\frac{𝐹 cos⁡𝛼 𝑑𝑠}{𝑑𝑡}=𝐹 𝑣 cos⁡𝛼
```
Мощность равна произведению касательной составляющей силы на скорость. Измеряется мощность в ваттах. 1 ватт=1 Дж/с.

## Теорема об изменении кинетической энергии точки
Кинетической энергией материальной точки называется скалярная величина
```math
𝑇=\frac{𝑚𝑣^2}{2}
```
равная половине произведения массы точки на квадрат ее скорости.
Единицей измерения кинетической энергии является **джоуль**.
**Изменение кинетической энергии точки** при некотором ее перемещении равно алгебраической сумме работ всех действующих на точку сил на том же перемещении.
```math
\frac{𝑚_2 𝑣_2^2}{2}−\frac{𝑚_1 𝑣_1^2}{2}=𝐴_{1,2}
```

## Потенциальная энергия

Область, во всех точках которой на материальную точку, в эту область помещенную, действует некоторая сила, называется **силовым полем** для данной силы.
Силовое поле является **потенциальным**, если
```math
\frac{𝜕𝑊}{𝜕𝑥}=−𝐹_𝑥,{\ \ \ \ }\frac{𝜕𝑊}{𝜕y}=−𝐹_𝑦,{\ \ \ \ }\frac{𝜕𝑊}{𝜕𝑧}=−𝐹_𝑧
```
В этом случае 𝑊(𝑥, 𝑦, 𝑧) называется **потенциалом** силы F, или **потенциальной энергией точки**.
