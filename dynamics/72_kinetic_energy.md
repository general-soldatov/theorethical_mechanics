# Эмпирическое определение моментов инерции

В ряде случаев для тел нестандартной формы необходимо экспериментально определять моменты или радиусы инерции. Для этого можно использовать метод маятниковых колебаний, сущность которого состоит в следующем.

У тела определяется центр масс, затем подвешивается в точке О на расстоянии ОС и замеряют период малых колебаний. Затем подвешивается математический маятник, который также совершает малые колебания. И в результате опыта подбирают длину маятника, соответствующую периоду физического маятника. После чего определим искомый радиус инерции:
```math
𝜌_𝑐=\sqrt{𝐿_{пр}\cdot 𝑂𝐶 −𝑂𝐶^2 }
```
Также может быть использован метод крутильных колебаний, откуда момент инерции можем найти по формуле:
```math
𝐽_𝑧=𝐽_{эт} \frac{𝑇^2}{𝑇_{э}^2}
```

# Плоское движение твердого тела

Плоское движение можно описать с помощью системы уравнений:
```math
\begin{cases} 𝑚\ddot{𝑥}=∑𝐹_{𝑖𝑥} \\ 𝑚\ddot{𝑦}=∑𝐹_{𝑖𝑦} \\ 𝐽_𝑐 \ddot{𝜑}=∑𝑚_𝑐(𝐹_𝑖)\end{cases}
```

![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/task24.png)

Пусть однородный диск катится вверх по наклонной плоскости с углом $\alpha$, имея начальную скорость центра $𝑣_0$. Коэффициент сопротивления качению равен 𝑓. Требуется определить, какой путь пройдет диск до остановки?

Выберем наклонную плоскость в качестве направления оси. На диск, кроме силы тяжести $mg$, действуют еще проекции нормальной реакции $𝑁_1$ и $𝑁_2$.

Запишем уравнения плоского движения диска:
```math
\begin{cases}𝑚\ddot{𝑥}=−𝑚𝑔 sin⁡𝛼−𝑁_2 \\ 𝐽_𝑐 \ddot{𝜑}=𝑁_2 𝑅−𝑁_1 𝜇\end{cases}
```
Здесь $R$ – радиус диска, $μ$ – коэффициент трения качения.

Учитывая, что в направлении, перпендикулярном оси $х$, движение отсутствует, получаем $𝑚𝑔 cos⁡𝛼=𝑁_1$, тогда:
```math
\frac{1}{2} 𝑚𝑅^2 \ddot{𝜑}=𝑁_2 𝑅−𝑁_1 𝜇 ⇒  \frac{1}{2} 𝑚𝑅\ddot{𝜑}=𝑁_2−𝑁_1 \frac{𝜇}{𝑅}
```
Обозначим $𝑓=\frac{𝜇}{𝑅} и выразим $𝑁_2$:
```math
𝑁_2=\frac{1}{2} 𝑚𝑅\ddot{𝜑}+𝑓𝑁_1=\frac{1}{2} 𝑚𝑅\ddot{𝜑}+𝑓𝑚𝑔 cos⁡𝛼
```
Подставим соотношение в (1) уравнение:
```math
𝑚\ddot{𝑥}=−𝑚𝑔 sin⁡𝛼−\frac{1}{2} 𝑚𝑅\ddot{𝜑}+𝑓𝑚𝑔 cos⁡𝛼
```
![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/meme36.png)

Упростим выражение с учётом $\ddot{𝑥}=𝑅\ddot{𝜑}$:
```math
\ddot{𝑥}−\frac{1}{2} \ddot{𝑥}=−𝑔 sin⁡𝛼+𝑓𝑔 cos⁡𝛼{\ \ \ }⇒{\ \ \ }\ddot{𝑥}=−\frac{2}{3}𝑔(sin⁡𝛼+𝑓 cos⁡𝛼 )
```
Решим задачу Коши при $𝑡=0$, $𝑥=0$, $\dot{𝑥}=𝑣_0$:
```math
\dot{𝑥}=−\frac{2}{3} 𝑔(sin⁡𝛼+𝑓 cos⁡𝛼 )𝑡+𝑣_0,{\ \ }𝑥=−\frac{1}{3} 𝑔(sin⁡𝛼+𝑓 cos⁡𝛼 ) 𝑡^2+𝑣_0 𝑡
```
Поскольку при полной остановке $\dot{𝑥}=0$, то найдём время движения:
```math
𝑡=\frac{3𝑣_0}{2𝑔(sin⁡𝛼+𝑓 cos⁡𝛼 )}
```
А пройденный путь равен:
```math
𝑥=−\frac{3𝑣_0^2 𝑔(sin⁡𝛼+𝑓 cos⁡𝛼 )}{4𝑔^2 (sin⁡𝛼+𝑓 cos⁡𝛼 )^2}+\frac{3𝑣_0^2}{2𝑔(sin⁡𝛼+𝑓 cos⁡𝛼 )} =\frac{3𝑣_0^2}{4𝑔(sin⁡𝛼+𝑓 cos⁡𝛼 )}
```
