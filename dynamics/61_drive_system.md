# Дифференциальные уравнения движения системы
Рассмотрим систему, состоящую из n точек. Выделим любую точку системы с массой $m_к$. Обозначим равнодействующую всех приложенных к точке внешних сил $𝐹_𝑘^𝑒$, а равнодействующую внутренних сил $𝐹_𝑘^𝑖$.
Если при этом точка имеет ускорение $𝑎_𝑘$, то по второму закону динамики:
```math
𝑚_𝑘 \vec{𝑎}_𝑘=\vec{𝐹_𝑘^𝑒}+\vec{𝐹_𝑘^𝑖}
```
Аналогичный результат получим для каждой точки. Следовательно, для всей системы будет:
```math
\begin{cases} 𝑚_1 \vec{𝑎_1}=\vec{𝐹_1^𝑒}+\vec{𝐹_1^𝑖} \\ 𝑚_2 \vec{𝑎_2}=\vec{𝐹_2^𝑒}+\vec{𝐹_2^𝑖} \\ ........... \\ 𝑚_𝑛 \vec{𝑎_𝑛}=\vec{𝐹_𝑛^𝑒}+\vec{𝐹_𝑛^𝑖} \end{cases}
```
Эта система представляет **дифференциальные уравнения движения системы в векторной форме**.
Спроецировав равенства на координатные оси, получим соответствующие уравнения.

Полное решение заключается в определении движения каждой точки системы, зная силы и наложенные связи.

Данная задача очень сложная, поэтому достаточно найти какие-то характеристики, определяющие движение системы в целом используя общие теоремы динамики.

## Теорема о движении центра масс
В ряде случаев для определения характера движения системы (например, твердого тела) требуется знать закон движения ее центра масс. Найдём его:
```math
∑𝑚_𝑘 \vec{𝑎_𝑘}=∑\vec{𝐹_𝑘^𝑒}+∑\vec{𝐹_𝑘^𝑖}
```
Преобразуем левую часть этого уравнения. Зная, что $𝑀\vec{𝑟_𝑐}=∑𝑚_𝑘\vec{𝑟_𝑘}$, найдём производную по времени с учётом правила: производная суммы равна сумме производных:
```math
𝑀 \frac{𝑑^2 \vec{𝑟_𝑐}}{𝑑𝑡^2}=∑𝑚_𝑘  \frac{𝑑^2 \vec{𝑟_𝑘}}{𝑑𝑡^2}   ⇒  𝑀\vec{𝑎_𝑐}=∑𝑚_𝑘\vec{𝑎_𝑘}
```
Где $\vec{𝑎_𝑐}$ - ускорение центра системы. С учётом этого и по первому свойству внутренних сил $∑\vec{𝐹_𝑘^𝑖}=0$, получим:
```math
𝑀\vec{𝑎_𝑐}=∑\vec{𝐹_𝑘^𝑒}
```
**Произведение массы системы на ускорение ее центра масс равно геометрической сумме всех действующих на систему внешних сил.**

![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/meme25.png)

Проектируя обе части полученной теоремы на координатные оси, получим:
```math
𝑀\ddot{𝑥_𝑐}=∑𝐹_{𝑘𝑥}^𝑒,{\ \ \ }𝑀\ddot{𝑦_𝑐}=∑𝐹_{𝑘𝑦}^𝑒,{\ \ \ }𝑀\ddot{𝑧_𝑐}=∑𝐹_{𝑘𝑧}^𝑒
```
Значение доказанной теоремы состоит в следующем:
* Решения, которые мы получаем, рассматривая тело как материальную точку, определяют закон движения центра масс этого тела.
* Теорема позволяет исключать из рассмотрения все неизвестные внутренние силы.

![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/meme26.png)

**Закон сохранения движения центра масс**

- Если $∑\vec{𝐹_𝑘^𝑒}=0$, то центр масс системы движется с постоянной по модулю и направлению скоростью.
- Если $∑𝐹_{𝑘𝑥}^𝑒=0$, то проекция скорости центра масс на данную ось постоянна.

Отсюда следует **закон сохранения положения центра масс системы**:
- Если $∑\vec{𝐹_𝑘^𝑒}=0$ и скорость центра масс в какой-то момент времени равна нулю, то положение центра масс системы не изменяется.
- Если $∑𝐹_{𝑘𝑥}^𝑒=0$ и проекция скорости центра масс на некоторую ось $x$ в какой-то момент времени равна нулю, то координата х центра масс системы не изменяется.

### Пример
Пусть тело массой $m$ скатывается по плоскости треугольника массой $M$, лежащего гладкой горизонтальной поверхности.

![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/task16.png)

Требуется определить, на сколько переместится треугольник в горизонтальном направлении (ось $x$), если указанное тело пройдет путь $L$. Так как в направлении движения внешние силы отсутствуют, и начальная скорость центра масс равна нулю, то координата $x$ центра масс не меняется, то есть:
```math
𝑀_𝑐  𝑥_𝑐=𝑐𝑜𝑛𝑠𝑡
```
Отсюда:
```math
𝑀𝑥_1+𝑚𝑥_2=𝑀𝑥_3+𝑚𝑥_4
```
где $x_1$ и $x_2$ – координаты центры масс тел в первом положении, $x_3$ и $x_4$ – координаты центры масс тел во втором положении. Обозначим через $Δ𝑥$ сдвиг треугольника вдоль оси $𝑥$ в отрицательном направлении. Тогда
```math
𝑥_3  = 𝑥_1  −Δ𝑥 ,{\ \ \ }𝑥_4= 𝑥_2−Δ𝑥 + 𝐿 cos⁡𝛼.
```
Подставляя последние выражения в формулу, получаем:
```math
𝑀𝑥_1+𝑚𝑥_2=𝑀(𝑥_1−Δ𝑥)+(𝑥_2−Δ𝑥+𝐿 cos⁡𝛼 )
```
Раскрываем скобки, получаем:
```math
0=−𝑀Δ𝑥−𝑚Δ𝑥+𝑚𝐿 cos⁡𝛼
```
Отсюда получим:
```math
Δ𝑥=\frac{𝑚𝐿 cos⁡𝛼}{𝑀+𝑚}
```
Если $𝑚→0$, то $Δ𝑥→0$, при малой массе тела, треугольник практически не перемещается. Если же $𝑀→0$, то $Δ𝑥→𝐿 cos⁡𝛼$, то есть при малой массе треугольника он перемещается на расстояние, равное горизонтальному перемещению тела.
