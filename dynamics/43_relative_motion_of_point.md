# Движение теряющего массу элемента по прямолинейной плоской поверхности

В процессе сельскохозяйственного производства часто встречают процессы сепарации сыпучих сред, смысл которых состоит в удалении ненужных примесей из очищаемого продукта.
Это нередко выполняется посредством движения некоторого потока по решетам с различного вида отверстиями, подобранными так, что ненужные примеси естественным образом удаляются из сепарируемой смеси.
В этом случае масса элемента смеси представляет некоторую функциональную зависимость, поэтому, используя свойство производной произведения двух функций, уравнение динамики можно записать в виде:
```math
𝑚 \frac{𝑑\vec{𝑣}}{𝑑𝑡}+\vec{𝑣}\frac{𝑑𝑚}{𝑑𝑡}=∑\vec{𝐹_𝑖}
```
В случае сепарации на прямолинейно наклонно расположенных решетах, последнее уравнение принимает вид:
```math
𝑚\frac{𝑑𝑣}{𝑑𝑡}=∑𝐹_𝑖−𝑣 \frac{𝑑𝑚}{𝑑𝑡}
```
Здесь $v$–скорость элемента массы вдоль оси z.
Учитывая, что сумма активных сил согласно уравнению равна:
```math
\ddot{𝑧}=𝑔(sin⁡𝜑−𝑓 cos⁡𝜑 )
```
Получим:
```math
m\ddot{𝑧}=𝑚𝑔(sin⁡𝜑−𝑓 cos⁡𝜑 )−𝑣𝑚 ̇
```
В данном процессе элемент массы можно вычислять по формуле:
```math
𝑚=𝑀+𝑚_{пр} 𝑒^{−𝜆𝑧}
```
Где $M$ – неизменяемая масса, $m_пр$ – начальная масса засорителя, 𝜆 – коэффициент сепарации.

![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/meme21.png)

Найдем производную по времени от массы элемента:
```math
\dot{𝑚}=\frac{𝑑𝑚}{𝑑𝑧}=\frac{𝑑𝑚}{𝑑𝑧}\frac{𝑑𝑧}{𝑑𝑡}=−𝜆𝑚_{пр} 𝑒^{−𝜆𝑧}\dot{𝑧}
```
Учитывая, что $𝑣=\dot{𝑧}$ и подставляя представленное соотношение в уравнение получим:
```math
m\dot{𝑧}=𝑚𝑔(sin⁡𝜑−𝑓 cos⁡𝜑 )−𝜆𝑚_{пр} 𝑒^{−𝜆𝑧}\dot{𝑧}^2
```
Разделим уравнение на массу:
```math
\ddot{𝑧}=𝑔(sin⁡𝜑−𝑓 cos⁡𝜑 )−\frac{𝜆𝑚_{пр} 𝑒^{−𝜆𝑧}\dot{𝑧}^2}{𝑀+𝑚_{пр} 𝑒^{−𝜆𝑧}}
```
Обычно засоренность определяют отношением массы засорителей к массе чистой фракции $𝑝=\frac{𝑚_{пр}}{𝑀}$, поэтому уравнение окончательно примет вид:
```math
\ddot{𝑧}=𝑔(sin⁡𝜑−𝑓 cos⁡𝜑 )−\dot{𝑧}^2 \frac{𝜆𝑝𝑒^{−𝜆𝑧}}{1+𝑝𝑒^{−𝜆𝑧}}
```
Это также квазилинейное дифференциальное уравнение.

## Движение точки в поле силы тяжести с учетом сопротивления воздуха
В большинстве задач прикладной механики аналитическое решение задач движения невозможно. Из предыдущего примеры мы видели, что это возникает в случае, когда дифференциальные уравнения нелинейны или квазилинейны.
![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/task8.png)

Рассмотрим ещё один пример, где решение задачи сводится к решению квазилинейной системы уравнений. Имеется плоское движение в поле силы тяжести с учётом сопротивления воздуха.
На эту частицу М массой 𝑚, движущейся по траектории 𝐿 с текущей скоростью 𝑣 действуют сила тяжести $m\vec{𝑔}$ и сила сопротивления воздуха $\vec{𝐹_𝑐}=𝑚𝑘𝑣^2 \vec{𝜏}$, направленная по касательной к траектории и противоположно вектору скорости.
Постоянная 𝑘 называется **коэффициентом парусности частицы**, размерность $1/м$, зависящей от формы частицы.
Дифференциальные уравнения движения в проекциях на оси координат имеют вид:
```math
\begin{cases}𝑚\ddot{𝑥}=−𝑚𝑘𝑣^2 cos⁡𝜑  \\ 𝑚\ddot{𝑦}=−𝑚𝑔−𝑚𝑘𝑣^2  sin⁡𝜑 \end{cases}
```
Учтём, что $cos⁡𝜑=\frac{𝑣_𝑥}{𝑣}$, а $sin⁡𝜑=\frac{𝑣_𝑦}{𝑣}$, получим
```math
\begin{cases}\ddot{𝑥}=−𝑘𝑣_𝑥 𝑣    \\ \ddot{𝑦}=−𝑔−𝑘𝑣_𝑦 𝑣\end{cases}
```
Зная, что
```math
𝑣=\sqrt{𝑣_𝑥^2+𝑣_𝑦^2}
```
А $𝑣_𝑥=\dot{𝑥}$, $𝑣_𝑦=\dot{𝑦}$, получим систему уравнений, описывающих рассматриваемое движение:
```math
\begin{cases}\ddot{𝑥}=−𝑘\dot{𝑥}\sqrt{\dot{𝑥}^2+\dot{𝑦}^2}  \\ \ddot{𝑦}=−𝑔−𝑘\dot{𝑦}\sqrt{\dot{𝑥}^2+\dot{𝑦}^2} \end{cases}
```
Добавим к этой задаче н.у. $𝑡=0$,  $𝑥=0$, $\dot{𝑥} =𝑣_0  cos⁡𝛼$, $𝑦=ℎ$,  $\dot{𝑦}=𝑣_0 sin⁡𝛼$ получим задачу для определения закона движения частицы.
В силу квазилинейности системы, её решение возможно только численными методами. Решение данной задачи позволяет определять характер движения частиц с различными коэффициентами парусности.
