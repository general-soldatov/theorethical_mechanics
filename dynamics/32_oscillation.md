# Свободные колебания в среде с сопротивлением
Часто в реальных условиях на точку, помимо восстанавливающей силы, действует еще и сила сопротивления, направленная в сторону, противоположную движению точки.
В случае малых скоростей колебаний:
```math
𝑅=−𝜇𝑣
```
Получим дифференциальные уравнения колебаний точки.
```math
𝑚\ddot{𝑥}=−𝑐𝑥−𝜇\dot{𝑥}
```
Разделим обе части уравнения на $𝑚$ с учётом, что $\frac{𝜇}{𝑚}=2𝛽$
```math
\ddot{𝑥}+2𝛽\dot{𝑥}+𝜔^2 𝑥=0
```
Последнее уравнение представляет собой **дифференциальное уравнение свободных колебаний** при сопротивлении, пропорциональном скорости.

![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/meme9.png)

Решение этого уравнения ищут в зависимости от соотношения коэффициентов $𝛽$ и $𝜔$. Перейдём к характеристическому уравнению:
```math
𝑘^2+2𝛽𝑘+𝜔^2=0
```
Корни данного уравнения имеют вид:
```math
𝑘=−𝛽±\sqrt{𝛽^2−𝜔^2}
```
заменим $𝜔_1=\sqrt{𝛽^2−𝜔^2}$.

В случае $𝛽<𝜔$ (случай малого сопротивления) восстанавливающая сила гораздо больше силы сопротивления. Корень характеристического уравнения является комплексным, поэтому решение выглядит следующим образом:
```math
𝑥=𝑒^{−𝛽𝑡} (𝐶_1  cos(𝜔_1 𝑡)+𝐶_2  sin⁡(𝜔_1 𝑡))
```
Подставляя $𝐶_1=𝐴 sin⁡𝜑$ и $𝐶_2=𝐴 cos⁡𝜑$. Получим:
```math
𝑥=𝐴𝑒^{−𝛽𝑡} sin⁡(𝜔_1 𝑡+𝜑)
```
Уравнение показывает, что в этом случае наблюдаются затухающие колебания, поскольку:
```math
\lim_{𝑡→∞}⁡𝑥=0
```
В случае $𝛽>𝜔$ (случай большого сопротивления) восстанавливающая сила гораздо меньше силы сопротивления.

Корень характеристического уравнения является действительным, поэтому решение выглядит следующим образом:
```math
𝑥=𝐶_1 𝑒^{−(𝛽+𝜔_1)𝑡}+𝐶_2 𝑒^{−(𝛽−𝜔_1)𝑡}
```
Если $𝛽=𝜔$ (восстанавливающая сила соразмерна с силой сопротивления) дискриминант характеристического уравнения равен 0, поэтому общее решение уравнения имеет вид:
```math
𝑥=𝑒^{−𝛽𝑡}(𝐶_1+𝐶_2 𝑡)
```
В последних двух случаях вместо колебаний наступает так называемое **апериодическое движение**.
Эти случаи хорошо иллюстрируются работой пружины (затухающие колебания) и амортизатора (апериодическое движение) в подвеске большинства автомобилей.

# Вынужденные колебания
Рассмотрим важный случай колебаний, возникающих, когда на точку, кроме восстанавливающей силы $F$, действует еще и периодически изменяющаяся со временем сила $Q$, проекция которой на ось $x$ равна:
```math
𝑄=𝑄_0  sin⁡(𝜔_𝑒 𝑡)
```
Эта сила называется **возмущающей**, а колебания, происходящие под действием такой силы, называют **вынужденными**. Напишем основное уравнение динамики:
```math
𝑚\ddot{𝑥}=−𝑐𝑥−𝜇\dot{𝑥}+𝑄_0  sin⁡(𝜔_𝑒 𝑡)
```
Разделим обе части уравнения на $m$. При этом $𝑄_0/𝑚=𝑎_𝑒$, тогда:
```math
\ddot{𝑥}+2𝛽\dot{𝑥}+𝜔^2 𝑥=𝑎_𝑒  sin(𝜔_𝑒 𝑡)
```
Последнее уравнение является **дифференциальным уравнением вынужденных колебаний точки**. Поскольку уравнение является линейным неоднородным, то, согласно теореме, общее решение неоднородного уравнения $𝑥_{он}$ равно сумме общего решения соответствующего однородного уравнения $𝑥_{𝑜𝑜}$ и любого частного решения $𝑥_{чн}$ исходного неоднородного уравнения:
```math
𝑥_{он}=𝑥_{𝑜𝑜}+𝑥_{чн}
```
В случае свободных колебаний с малым сопротивлением решение $𝑥_{𝑜𝑜}$ было найдено ранее в виде:
```math
𝑥_{𝑜𝑜}=𝐴𝑒^{−𝛽𝑡} sin⁡(𝜔_1 𝑡+𝜑)
```
Частное решение неоднородного уравнения ищется в виде $𝐵 sin(𝜔_𝑒 𝑡)$. Подставим частное решение в основное уравнение:
```math
−𝐵𝜔_𝑒^2  sin⁡(𝜔_𝑒 𝑡)+2𝛽𝜔_𝑒 𝐵 cos(𝜔_𝑒 𝑡)+𝜔^2 𝐵 sin⁡(𝜔_𝑒 𝑡)=𝑎_𝑒  sin⁡(𝜔_𝑒 𝑡)
```
![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/meme11.png)

Сократим уравнение на $sin(𝜔_𝑒 𝑡)$:
```math
−𝐵𝜔_𝑒^2+2𝛽𝜔_𝑒 𝐵 \cdot tg(𝜔_𝑒 𝑡)+𝜔^2 𝐵=𝑎_𝑒
```
Отсюда получим 𝐵:
```math
𝐵=\frac{𝑎_𝑒}{𝜔^2−𝜔_𝑒^2+2𝛽𝜔_𝑒 tg(𝜔_𝑒 𝑡)}
```
Таким образом, решение уравнения принимает вид:
```math
𝑥=𝐴𝑒^{−𝛽𝑡}  sin⁡(𝜔_1 𝑡+𝜑)+\frac{𝑎_𝑒 sin⁡(𝜔_𝑒 𝑡)}{𝜔^2−𝜔_𝑒^2+2𝛽𝜔_𝑒 tg(𝜔_𝑒 𝑡)}
```
В случае, когда 𝛽=0, решение примет вид:
```math
𝑥=𝐴 sin⁡(𝜔_1 𝑡+𝜑)+\frac{𝑎_𝑒  sin⁡(𝜔_𝑒 𝑡)}{𝜔^2−𝜔_𝑒^2}
```
Постоянные $А$ и $𝜑$ могут быть найдены из начальных условий.

Как следует из решения при $𝜔_𝑒→𝜔$ величина второго слагаемого стремится к бесконечности, то есть если частота вынуждающей силы стремится к частоте собственных колебаний, то возникает условие **резонанса**, или неограниченного роста амплитуды колебаний.

## Задача
Рассмотрим пример движения математического маятника, то есть тяжелой материальной точки, подвешенной на невесомой нерастяжимой нити длиной $L$. На точку действуют сила тяжести $mg$ и сила натяжения нити $T$.

![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/task4.png)


По теореме об изменении количества движения:
```math
\frac{𝑑}{𝑑𝑡}\vec{𝑚}_0(𝑚\vec{𝑣})=\vec{𝑚_0}(\vec{𝐹})
```
Как видно из рисунка, векторы сил и вектор скорости лежат в одной плоскости, поэтому последнее уравнение примет вид:
```math
\frac{𝑑}{𝑑𝑡}(𝑚𝑣𝐿)=−𝑚𝑔𝐿 sin⁡𝜑
```
Сократим уравнение на $𝑚𝐿$:
```math
\frac{𝑑𝑣}{𝑑𝑡}=−𝑔 sin⁡𝜑
```
Примем, что $𝑣=𝜔𝐿=\dot{𝜑}𝐿$, получим:
```math
\ddot{𝜑}𝐿+𝑔 sin⁡𝜑=0
```
В случае малых колебаний $sin⁡𝜑≈𝜑$ и последнее уравнение принимает вид свободных колебаний, аналогичное дифференциальному уравнению колебаний:
```math
\ddot{𝜑}+\frac{𝑔}{𝐿} 𝜑=0
```
Таким образом, величина $\frac{g}{L}$ аналогична $𝜔^2$, откуда следует, что периодом колебаний математического маятника является величина:
```math
𝑇=2𝜋\sqrt{\frac{𝐿}{𝑔}}
```
