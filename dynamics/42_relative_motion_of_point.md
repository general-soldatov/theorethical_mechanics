# Движение точки по плоской кривой
Рассмотрим движение точки по некоторой гладкой кривой: $𝑦=𝑓(𝑥)$ под действием силы тяжести.
Отметим, что на точку при движении действуют три активные силы: сила тяжести $𝑚𝑔$, нормальная реакция $𝑁$ и сила трения $𝐹_тр  =𝑓𝑁$.
Тогда уравнение движения точки в проекции на ось $x$ имеет вид:
```math
𝑚\ddot{𝑥}=𝑁 cos⁡𝛼−𝐹_{тр} sin⁡𝛼
```
Иначе:
```math
𝑚\\ddot{𝑥}=𝑁(cos⁡𝛼−𝑓 sin⁡𝛼 )
```
Нормальная реакция $N$ определяется условием равновесия сил вдоль оси, направленной по нормали к поверхности:
```math
𝑁=𝑚𝑔 sin⁡𝛼+\frac{𝑚𝑣^2}{𝜌}
```
![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/task6.png)

Последнее слагаемое в формуле для определения нормальной реакции является центробежной силой, с которой точка действует на направляющую кривую.
В одном из разделов высшей математики – дифференциальной геометрии – выводится формула для определения радиуса кривизны:
```math
𝜌=\frac{\left[1+(𝑦_𝑥')^2 \right]^{3/2}}{(𝑦_𝑥'')}
```
Вспомним школьные тригонометрические формулы:
```math
cos⁡𝛼=\frac{ctg⁡𝛼}{\sqrt{1+ctg^2⁡𝛼}},{\ \ \ }sin⁡𝛼=\frac{1}{\sqrt{1+ctg^2⁡𝛼}}
```
Учитывая, что:
```math
tg⁡𝛽=𝑦_𝑥'=−tg\left(⁡{\frac{𝜋}{2}−𝛼}\right)=−ctg⁡𝛼.
```
Откуда  $ctg⁡𝛼=−𝑦_𝑥'$, получаем:
```math
cos⁡𝛼=\frac{−𝑦_𝑥'}{\sqrt{1+(𝑦_𝑥')^2}},{\ \ \ \ }sin⁡𝛼=\frac{1}{\sqrt{1+(𝑦_𝑥')^2}}
```
Отметим также, что
```math
𝑣^2=𝑣_𝑥^2+𝑣_𝑦^2=\dot{𝑥}^2+\dot{𝑦}^2=\dot{𝑥}^2+\left(\frac{𝑑𝑦}{𝑑𝑡}\right)^2=\dot{𝑥}^2+\left(\frac{𝑑𝑦}{𝑑𝑥}\frac{𝑑𝑥}{𝑑𝑡}\right)^2=\dot{𝑥}^2+(𝑦_𝑥' \dot{𝑥})^2
```
В окончательном виде:
```math
𝑣^2=\dot{𝑥}^2 [1+(𝑦_𝑥')^2 ]
```
Запишем исходное уравнение:
```math
𝑚\ddot{𝑥}=𝑚\left(𝑔 sin⁡𝛼+\frac{𝑣^2}{𝜌}\right)(cos⁡𝛼−𝑓 sin⁡𝛼 )⇒\ddot{𝑥}=\left(𝑔 sin⁡𝛼+\frac{𝑣^2}{𝜌}\right)(cos⁡𝛼−𝑓 sin⁡𝛼 )
```
Преобразуем его с учётом новых выражений:
```math
\ddot{𝑥}=\left(𝑔 \frac{1}{\sqrt{1+(𝑦_𝑥')^2}}+\frac{\ddot{𝑥}^2 𝑦_𝑥''}{\sqrt{1+(𝑦_𝑥')^2}}\right)\left(\frac{−𝑦_𝑥'}{\sqrt{1+(𝑦_𝑥')^2}}−𝑓 \frac{1}{\sqrt{1+(𝑦_𝑥')^2}}\right)
```
### Задача Коши
После очевидных преобразований получаем:
```math
\ddot{𝑥}=−\frac{(𝑔+\dot{𝑥}^2 𝑦_𝑥'')(𝑓+𝑦_𝑥')}{1+(𝑦_𝑥')^2 }
```
Добавляя к полученному уравнению начальные условия вида:
$𝑡=0$, $𝑥(0)=𝑥_0$, $\dot{𝑥}(0)=𝑣_{𝑥0}$, получаем задачу Коши для квазилинейного дифференциального уравнения, которое имеет только численное решение.

Не являющееся линейным дифференциальное уравнение называется **квазилинейным**, если его старшую производную можно выразить через младшие. В противном случае уравнение нелинейно.

## Движение по наклонной плоскости
Рассмотрим частный случай движения по наклонной плоскости. В этом случае:
```math
tg⁡𝛽=𝑦_𝑥'=−tg⁡𝜑,{\ \ \ }𝑦_𝑥''=0
```

![Image](https://github.com/general-soldatov/theorethical_mechanics/blob/main/images/task7.png)

С учётом тождества:
```math
1+tg^2⁡𝜑=\frac{1}{cos^2⁡𝜑}
```
Исходное уравнение примет вид:
```math
\ddot{𝑥}=−𝑔(𝑓−tg⁡𝜑 )  cos^2⁡𝜑=𝑔(sin⁡𝜑−𝑓 cos⁡𝜑 )  cos⁡𝜑
```
Поскольку $𝑥=𝑧 cos⁡𝜑$, то $\ddot{𝑥}=\ddot{𝑧} cos⁡𝜑$, откуда:
```math
\ddot{𝑧}=𝑔(sin⁡𝜑−𝑓 cos⁡𝜑 )
```
Что и является дифференциальным уравнением движения точки по наклонной плоскости.
