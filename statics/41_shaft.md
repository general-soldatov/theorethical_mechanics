# Стержневые конструкции
**Фермы** –это геометрически неизменяемые системы прямолинейных стержней, соединенных шарнирами.  
Простейшая ферма – это ферма, имеющая наименьшее количество стержней при заданном количестве шарниров, называемых **узлами**.  
Соотношение между количеством стержней n и узлов s в простейших фермах. Поскольку основой построения простейшей фермы является треугольник АВС, (три шарнира – три стержня), то для присоединения каждого следующего шарнира потребуются два стержня:
```math
𝑛 = 3 + 2(𝑠 – 3) = 2𝑠 – 3 
```
