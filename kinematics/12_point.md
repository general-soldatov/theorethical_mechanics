# Определение скорости точки при задании её движения векторным способом

**Скорость** – это векторная величина, характеризующая быстроту и направление движения точки в данной системе отсчета.  
Пусть в момент времени t точка занимает положение М, определяемое радиусом-вектором r, а в момент t1=t+Δt она занимает положение М1 с радиусом вектором r1. Из треугольника ОММ1: 

  <img width="650" height="451" alt="image" src="https://github.com/user-attachments/assets/e17b035f-a5e0-4f13-bac1-87cc90110456" />  

```math
\vec{OM}_1 = \vec{OM}+\vec{MM}_1
```
При перемещении точки ее радиус-вектор получает приращение:
```math
\vec{r}_1=\vec{r}+\Delta \vec{r}
```
