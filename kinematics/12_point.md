# Определение скорости точки при задании её движения векторным способом

**Скорость** – это векторная величина, характеризующая быстроту и направление движения точки в данной системе отсчета.  
Пусть в момент времени t точка занимает положение М, определяемое радиусом-вектором r, а в момент t1=t+Δt она занимает положение М1 с радиусом вектором r1. Из треугольника ОММ1: 

  <img width="650" height="451" alt="image" src="https://github.com/user-attachments/assets/e17b035f-a5e0-4f13-bac1-87cc90110456" />  

```math
\vec{OM}_1 = \vec{OM}+\vec{MM}_1
```
При перемещении точки ее радиус-вектор получает приращение:
```math
\vec{r}_1=\vec{r}+\Delta \vec{r}
```
Отношение вектора перемещения Δ𝑟 к промежутку времени Δ𝑡, в течение которого произошло это перемещение, представляет собой вектор средней скорости 𝑣_𝑐𝑝  движения точки по хорде ММ_1:
```math
\vec{v}_{ср.} = \frac{\Delta \vec{r}}{\Delta t}
```
Однако данное движение возможно лишь в случае очень малого расстояния ММ1, т.е. при уменьшении промежутка времени Δ𝑡 и приближении его к нулю вектор Δ𝑟 также стремится к нулю, а вектор 𝑣_𝑐𝑝 – к некоторому пределу. Этот предел является вектором скорости точки в момент времени t:  
```math
\vec{v}=\lim_{\Delta t \to 0}\frac{\Delta\vec{r}}{\Delta t} = \frac{d\vec{r}}{dt}
```
Вектор скорости точки в данный момент времени равен производной от радиуса-вектора точки по времени и направлен по касательной к траектории в сторону движения точки.`$\displaystyle{\lim_{x \to \infty}}$`
