# Плоскопараллельное (плоское) движение твердого тела
**Плоским** движением твердого тела называется такое движение, при котором каждая точка тела движется параллельно некоторой фиксированной плоскости.  
<img width="686" height="608" alt="image" src="https://github.com/user-attachments/assets/71f214a4-1261-48b8-91ec-b32a4ae6a182" />  
  
Тело движется вокруг неподвижной плоскости П, $𝑀𝑀_1⊥П$. При движении этот отрезок остаётся параллельным своему начальному положению, следовательно все точки этого отрезка движутся тождественно и параллельно. При этом
```math
𝑣_𝑀=𝑣_{𝑀_{1}}=𝑣,{\ \ \ }𝑎_𝑀=𝑎_{𝑀_1}=𝑎
```
Следовательно, для изучения движения всего тела достаточно изучить движение одного из его сечений, параллельных фиксированной плоскости.  
Положение сечения в плоскости Оху определяется положением какого-нибудь проведенного в нем отрезка АВ. В свою очередь, положение этого отрезка можно определить, зная координаты $х_а$ и $у_а$ точки А и угол φ, который отрезок АВ образует с осью Ох. Точку А, выбранную для определения положения сечения S в дальнейшем будем называть **полюсом**.  
<img width="769" height="436" alt="image" src="https://github.com/user-attachments/assets/b2ca3f3d-edd0-4448-a113-e12c7a86e52e" />  
  
При движении фигуры величины $х_a$, $у_a$ и $\varphi$ будут изменяться с течением времени. Значит, чтобы знать закон движения отрезка АВ:
```math
𝑥_𝑎=𝑓_1 (𝑡){\ \ \ }𝑦_𝑎=𝑓_2 (𝑡){\ \ \ }𝜑=𝑓_3 (𝑡)
```
или **уравнениями движения плоской фигуры в ее плоскости**. Они же являются уравнениями плоского движения твердого тела.  
  
<img width="596" height="550" alt="image" src="https://github.com/user-attachments/assets/be15c287-c0ed-4b2c-996c-903b4f698bfc" />  

Положение любой точки М по отношению к осям Оху определяется радиус-вектором $\vec{𝑟}=\vec{𝑟}_𝐴 + \vec{𝑟}_1$, тогда  
```math
\vec{𝑣}_𝑀=\frac{𝑑\vec{𝑟}}{𝑑𝑡}=\frac{𝑑\vec{𝑟}_𝐴}{𝑑𝑡}+\frac{𝑑\vec{𝑟}_1}{𝑑𝑡}
```
Здесь $\frac{𝑑\vec(𝑟_𝐴}{𝑑𝑡}=\vec{𝑣}_А$ - скорость полюса А, $\frac{𝑑\vec{𝑟}_1}{𝑑𝑡}=\vec{𝑣}_МА$ - скорость, которую получает точка при $\vec{𝑣}_А =0$, следовательно:
```math
\vec{𝑣}_М =\vec{𝑣}_А+\vec{𝑣}_{МА}
```
При этом скорость, которую точка будет получать при вращении вокруг полюса А, будет
```math
\vec{𝑣}_{𝑀А}=\vec{𝜔}\cdot 𝑀𝐴
```
Таким образом, скорость любой точки М плоской фигуры геометрически складывается из скорости полюса и скорости, которую точка получает при вращении фигуры вокруг этого полюса.

### Теорема о проекциях скоростей двух точек тела
Проекции скоростей двух точек твердого тела на ось, проходящую через эти точки, равны друг другу.  
<img width="717" height="527" alt="image" src="https://github.com/user-attachments/assets/47704fcb-f12e-4aff-a5f4-2b435d451e43" />  

Рассмотрим какие-нибудь две точки А и В плоской фигуры. Принимая за полюс точку А, получим $\vec{𝑣_В}=\vec{𝑣_А} +\vec{𝑣_{ВА}}$. Проектируя обе части равенства на ось, направленную вдоль АВ и учитывая, что вектор $\vec{𝑣}_{ВА}$ перпендикулярен АВ, находим 
```math
𝑣_В\cdot cos⁡𝛽= 𝑣_𝐴 \cdot cos⁡𝛼
```
**Теорема доказана.**

### Определение скоростей точек плоской фигуры с помощью мгновенного центра скоростей
**Мгновенным центром скоростей** называется точка плоской фигуры, скорость которой в данный момент времени равна нулю.  
<img width="683" height="551" alt="image" src="https://github.com/user-attachments/assets/2c973abd-bd30-48d9-a3ae-e66834517551" />  

В случае непоступательного движения тела такая точка в каждый момент времени существует и она единственная.  
Скорости точек плоской фигуры определяются в данный момент времени так, как если бы движение фигуры было вращением вокруг мгновенного центра скоростей.
```math
𝑣_𝐴=𝜔 \cdot 𝑃𝐴;{\ \ \ }𝑣_𝐵=𝜔 \cdot 𝑃𝐵{\ \ }→{\ \ \ }\frac{𝑣_𝐴}{𝑃𝐴}=\frac{𝑣_𝐵}{𝑃𝐵}=𝜔
```
