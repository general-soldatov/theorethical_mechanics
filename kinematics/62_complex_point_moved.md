# Теорема о сложении ускорений (теорема Кориолиса)
Найдём ускорение точки при сложном движении.
```math
\vec{𝑎}_{абс}=\frac{𝑑\vec{𝑣_{абс}}}{𝑑𝑡}=\frac{𝑑\vec{𝑣_{отн}}}{𝑑𝑡}+\frac{𝑑\vec{𝑣_{пер}}}{𝑑𝑡}
```
Производные определяют изменение скоростей при абсолютном движении. Изменения складываются из изменений при относительном и переносном движениях.  
Введём индексы:  
1 – изменение скорости при относительном движении,  
2 –изменение скорости при переносном движении:  
```math
\vec{𝑎_{абс}}=\frac{𝑑\vec{𝑣_{отн1}}}{𝑑𝑡}+\frac{𝑑\vec{𝑣_{пер1}}}{𝑑𝑡}+\frac{𝑑\vec{𝑣_{отн2}}}{𝑑𝑡}+\frac{𝑑\vec{𝑣_{пер2}}}{𝑑𝑡}
```

<img width="598" height="587" alt="image" src="https://github.com/user-attachments/assets/80b68c5e-239c-4d06-b664-016a811a9efd" />

  
Относительное ускорение характеризует изменение относительной скорости только при относительном движении:
```math
\vec{𝑎_{отн}}=\frac{𝑑\vec{𝑣_{отн1}}}{𝑑𝑡}
```
Переносное ускорение характеризует изменение переносной скорости только при переносном движении
```math
\vec{𝑎_{пер}}=\frac{𝑑\vec{𝑣_{пер2}}}{𝑑𝑡}
```
В результате равенство примет вид:
```math
\vec{𝑎_{абс}}=\vec{𝑎_{отн}}+\vec{𝑎_{пер}}+\frac{𝑑\vec{𝑣_{пер1}}}{𝑑𝑡}+\frac{𝑑\vec{𝑣_{отн2}}}{𝑑𝑡}
```

## Кориолисово ускорение
Величина $𝑎_{кор}$ , характеризующая изменение относительной скорости точки при переносном движении и переносной скорости при ее относительном движении, называется **поворотным**, или **кориолисовым**, ускорением точки. 
```math
\vec{𝑎_{кор}}=\frac{𝑑\vec{𝑣_{пер1}}}{𝑑𝑡}+\frac{𝑑\vec{𝑣_{отн2}}}{𝑑𝑡}
```
С учётом этого окончательно получаем:
```math
\vec{𝑎_{абс}}=\vec{𝑎_{отн}}+\vec{𝑎_{пер}}+\vec{𝑎_{кор}}
```
При сложном движении ускорение точки равно геометрической сумме трех ускорений: **относительного**, **переносного** и **кориолисова**.  
Определим $\frac{𝑑\vec{𝑣_{отн2}}}{𝑑𝑡}$. Переносное движение складывается из поступательного движения полюса m и вращательного движения вокруг этого полюса 𝜔.  
<img width="719" height="719" alt="image" src="https://github.com/user-attachments/assets/0a269aa3-c540-4dcc-9334-768412c54db0" />  
  
При этом движении вектор $𝑣_{отн}$ переместится вместе с кривой АВ поступательно и одновременно повернётся вокруг точки $m_1$ до положения $m_1b_1$. 
В результате приращение вектора $𝑣_{отн}$ в переносном движении будет равно:
```math
d\vec{𝑣_{отн2}}=\vec{𝑏𝑏_1}=\vec{𝑣_𝑏}𝑑𝑡
```
где $\vec{𝑣_𝑏}$ - скорость, с которой перемещается точка b при повороте вектора $𝑣_{отн}$. Т.к. этот поворот происходит с угловой скоростью, то 
```math
\vec{𝑣_𝑏}=\vec{𝜔}×\vec{𝑚_1 𝑏}
```
отсюда $\vec{𝑣_{отн}}= \vec{𝑚_1 𝑏}$, получим:
```math
d\vec{𝑣_{отн2}}=\vec{𝜔}×\vec{𝑣_{отн}}𝑑𝑡
```
Или  
```math
\frac{d\vec{𝑣_{отн2}}}{𝑑𝑡}=\vec{𝜔}×\vec{𝑣_{отн}}
```
Теперь найдём $\frac{𝑑\vec{𝑣_{пер1}}}{𝑑𝑡}$.  
<img width="658" height="694" alt="image" src="https://github.com/user-attachments/assets/43ebbaa1-5a98-4c87-ad04-5e8fc001e407" />
  
При рассматриваемом переносном движении скорость любой точки на кривой АВ равна 
```math
\vec{𝑣_{пер}}=\vec{𝑣_0}+\vec{𝜔}×\vec{𝑟}
```
где $𝑣_0$ - скорость полюса. За промежуток времени dt точка совершит новое перемещение $\vec{𝑣_{отн}}𝑑𝑡=\vec{𝑀𝑀\`}$. Новое положение точки определяется радиус вектором $\vec{𝑟\`}=\vec{𝑟}+\vec{𝑀𝑀}\`$. При этом новая скорость точки определяется как
```math
\vec{𝑣'_{пер}}=\vec{𝑣_0}+\vec{𝜔}×\vec{𝑟}+\vec{𝑀𝑀'}
```
Тогда приращение вектора переносной скорости вследствие относительного движения будет равно
```math
𝑑\vec{𝑣_{пер1}}=\vec{𝑣' _{пер}}−\vec{𝑣_{пер}}=\vec{𝑣_0}+\vec{𝜔}×(\vec{𝑟}+\vec{𝑀𝑀'})−\vec{𝑣_0}−\vec{𝜔}×\vec{𝑟}
```
```math
𝑑\vec{𝑣_{пер1}}=\vec{𝜔}×\vec{𝑀𝑀'}=\vec{𝜔}×\vec{𝑣_{отн}}𝑑𝑡
```
Или
```math
\frac{𝑑\vec{𝑣_{пер1}}}{𝑑𝑡}=\vec{𝜔}×\vec{𝑣_{отн}}
```
<img width="730" height="547" alt="image" src="https://github.com/user-attachments/assets/0ca43789-6c35-4609-a0ed-33bd6a7b6b97" />  
  
**Кориолисово ускорение** равно удвоенному векторному произведению переносной угловой скорости на относительную скорость точки. 
```math
\vec{𝑎_{кор}}=2(\vec{𝜔}×\vec{𝑣_{отн}})
```
Модуль кориолисова ускорения
```math
𝑎_{кор}=2\cdot 𝜔\cdot 𝑣_{отн} sin⁡𝛼
```
где 𝛼 – угол между векторами $\vec{𝜔}$ и $\vec{𝑣_{отн}}$
Кориолисово ускорение равно нулю в трех случаях:  
* Если $\vec{𝜔}_{пер}=0$, т.е. в случае поступательного переносного движения.
* Если $\vec{𝑣}_{отн}=0$, т.е. в случае относительного покоя точки.
* Если $sin⁡𝛼=0$, т.е. когда относительная скорость точки параллельна оси вращения.

### Направление кориолисова ускорения
Направлен вектор $\vec{𝑎}\_{кор}$, так же как и вектор $\vec{𝜔}×\vec{𝑣}\_{отн}$, т.е. перпендикулярно плоскости, проходящей через векторы $\vec{𝑣}\_{отн}$ и $\vec{𝜔}$, в ту сторону, откуда кратчайшее совмещение этих векторов видно происходящим против хода часовой стрелки.  

Для определения направления кориолисова ускорения удобно пользоваться **правилом Жуковского**: чтобы найти направление кориолисова ускорения, следует спроектировать относительную скорость точки на плоскость, перпендикулярную оси переносного вращения, и повернуть эту проекцию в той же плоскости на 90° в сторону переносного вращения.
