# Определение скорости точки при задании её движения координатным способом

Пусть движение точки задано уравнениями $х=f_1(t)$, $у=f_2(t)$, $z=f_3(t)$.  
Обозначим через $\vec{𝑖}$, $\vec{𝑗}$,$\vec{𝑘}$ - орты осей координат. Проведем из начала координат О в движущуюся точку М радиус-вектор r: 
```math
\vec{OM}=\vec{r}=\vec{OA}+\vec{AB}+\vec{BM}=\vec{i}x+\vec{j}y+\vec{k}z
```

Скорость точки равна производной от радиуса-вектора по времени:
```math
\vec{v}=\frac{d\vec{r}}{dt}=\vec{i}\frac{dx}{dt}+\vec{j}\frac{dy}{dt}+\vec{k}\frac{dz}{dt}
```
Построив из точки М прямоугольный параллелепипед, ребра которого будут параллельны осям координат, а диагональ совпадает со скоростью v , получим проекции v на оси координат х у z, равные алгебраическим величинам отрезков МA, МB, MC. Получим:
```math
\vec{v} = \vec{i}v_x+\vec{j}v_y+\vec{k}v_z
```
Отсюда:
```math
\displaystyle{\begin{case} v_x=\frac{dx}{dt} \\ v_y=\frac{dy}{dt} \\ v_z =\frac{dz}{dt} \end{case}}
```
Следовательно, проекции скорости точки на неподвижные оси декартовых координат равны первым производным от соответствующих координат по времени. Модуль:
𝑣=√(𝑣_𝑥^2+𝑣_𝑦^2+𝑣_𝑧^2 )
