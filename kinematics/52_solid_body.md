# Определение ускорений точек плоской фигуры
<img width="657" height="673" alt="image" src="https://github.com/user-attachments/assets/a8e2c7f7-a818-4351-8732-0f518e9a5677" />  
  
Скорость точки М: 
```math
\vec{𝑣_𝑀}=\frac{𝑑\vec{𝑟}}{𝑑𝑡}=\frac{𝑑\vec{𝑟_𝐴}{𝑑𝑡}+\frac{𝑑\vec{𝑟_1}}{𝑑𝑡}
```
Отсюда:
```math
\vec{𝑎_𝑀}=\frac{𝑑^2 \vec{𝑟}}{𝑑𝑡^2}=\frac{𝑑^2\vec{𝑟_𝐴}}{𝑑𝑡^2}+\frac{𝑑^2\vec{𝑟_1}}{𝑑𝑡^2}{\ \ \ →}\vec{𝑎_𝑀}=\vec{𝑎}_𝐴+\vec{𝑎_𝑀𝐴}
```
Значение $\vec{𝑎_{𝑀𝐴}}=\vec{𝑎_{𝑀𝐴}^𝜏}+\vec{𝑎_{𝑀𝐴}^𝑛}$, при этом:
```math
\vec{𝑎_𝑀}=\vec{𝑎_𝐴}+\vec{𝑎_{𝑀𝐴}^𝜏}+\vec{𝑎_{𝑀𝐴}^𝑛}
```
Ускорение любой точки М плоской фигуры геометрически складывается из ускорения какой-нибудь другой точки А, принятой за полюс, и ускорений, которые точка получает вследствие вращения вокруг этого полюса.
```math
\vec{𝑎_{𝑀𝐴}^𝜏}=𝜀 𝑀𝐴{\ \ \ \ \ }\vec{𝑎_{𝑀𝐴}^𝑛}=𝜔^2 𝑀𝐴
```
